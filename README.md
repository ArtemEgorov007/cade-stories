# Cade Stories

Демо-лендинг: образовательный портал о blockchain, ИИ и цифровой безопасности.

**Live:** https://artemegorov007.github.io/cade-stories/

---

## Зачем этот проект в портфолио

Это не рабочий оффер и не кейс с NDA — учебный сайт, через который видно, как я подхожу к **верстке под CPA и арбитраж в целом**:

- **Структура под трафик** — главная с понятным CTA, внутренние страницы (гайд, глосарий, блог), legal-блок (privacy, terms, cookies), контакт. Пользователь не упирается в тупик.
- **Доверие и «белый» контент** — нейтральная подача, без агрессивных обещаний; контент читается как редакционный, а не как «продажа в лоб».
- **Мобилка и десктоп** — один макет, ритм отступов, читаемость на 390px и 1280px.
- **Готовность к интеграции** — вёрстка отделена от логики: на GitHub Pages отдаётся статика; локально тот же проект собирается из PHP-шаблонов.

По сути — показ того, как я вижу **рабочий лендинг в арбитраже**: аккуратная оболочка, на которую поверх можно навесить трекинг, форму и оффер, не ломая UX.

## Что внутри

| Раздел | Содержание |
|--------|------------|
| Inicio | Hero, блоки доверия, три темы, превью статей |
| Aprender | Пошаговый гайд по трём направлениям |
| Glosario | 22 термина с фильтром по буквам |
| Blog | Листинг + 3 полноценные статьи |
| Legal | Privacidad, términos, cookies |
| Contacto | Форма, FAQ, контактные данные |

## Стек

- PHP (partials, шаблоны) → статический экспорт для GitHub Pages
- CSS: design tokens, тёмная тема, анимации, адаптив
- Vanilla JS: меню, reveal, cookie-banner, canvas на главной

## Локально

```bash
php -S localhost:8795 -t .
# открыть http://localhost:8795/index.html
```
